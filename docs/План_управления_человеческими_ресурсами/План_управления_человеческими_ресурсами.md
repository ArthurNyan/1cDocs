## 5. Набор команды

### 5.1 Процесс набора

1. **Определение потребностей** (2 дня)
   - Анализ требований к ролям
   - Определение необходимых компетенций

2. **Поиск кандидатов** (5 дней)
   - Внутренний поиск
   - Внешний рекрутинг

3. **Отбор** (3 дня)
   - Собеседования
   - Техническая оценка

4. **Назначение** (1 день)
   - Оформление
   - Ввод в проект
